## All-in-One Data Validation & Generation API

Stop paying for 5 separate APIs. DataForge combines **email validation, phone validation, URL validation, text analysis, QR code generation, hashing, and IP geolocation** into a single API with one key.

### ✅ Email Validation
Full email validation with MX record check, disposable email detection (140+ domains), free provider detection, role-based detection, and deliverability scoring (0-100).

### 📱 Phone Validation
International phone number validation with country detection, carrier info, line type, and E.164 formatting.

### 🔗 URL Validation
Validate URLs with optional live HTTP check to verify the endpoint responds.

### 📊 Text Analysis
Word count, sentence count, reading time, Flesch readability scores, sentiment analysis, and word frequency — all in one call.

### 🔐 Hashing & Security
10+ algorithms (MD5, SHA-256, SHA-512, BLAKE2b, BLAKE2s), HMAC generation, hash comparison, secure password generation, UUID v4, and cryptographic tokens.

### 📲 QR Code Generation
Generate QR codes as base64 PNG or direct image download. Customizable size (100-1000px).

### 🌍 IP Geolocation
Free IP-to-location lookup: country, city, region, timezone, ISP, coordinates.
